# 🏗️ Infrastructure

A Mimir fejlesztési és futtatási környezetének leírói.

## 🛠 Tartalom
* **Docker Compose:** Lokális fejlesztési környezet indításához (PostgreSQL, Redis, MinIO és a Mimir szolgáltatások).
* **Kubernetes (K8s):** HPA (Horizontal Pod Autoscaling) konfigurációk és manifest fájlok a felhős skálázáshoz.
* **Adatbázis sémák:** Inicializáló szkriptek (pl. a `FOR UPDATE SKIP LOCKED` sorhoz szükséges PostgreSQL táblákhoz).
