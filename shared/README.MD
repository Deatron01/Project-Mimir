# 🤝 Shared (Közös Erőforrások)

A Mimir mikroszolgáltatásai között megosztott típusok és sémák.

## 🛠 Tartalom
* **JSON Sémák:** A Heimdall által használt szigorú validációs sémák definíciói.
* **DTO-k (Data Transfer Objects):** A modulok közötti kommunikációhoz használt adatstruktúrák (pl. a The Forge és Bifrost között).
* **Közös segédfüggvények (Utils):** Keretrendszer-független apró logikák, amelyeket több szolgáltatás is használ.
